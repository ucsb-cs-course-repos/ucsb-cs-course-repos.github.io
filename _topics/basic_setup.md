# Configure the basic information for your course in `_config.yml`:

Example:

```yml
course: "CS56"
qtr: "F17"
quarter: "Fall 2017"
piazza_url: https://piazza.com/class/j80k1o0zsxg1j7
gauchospace_course_page: https://gauchospace.ucsb.edu/courses/course/view.php?id=19139
```
